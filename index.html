Autovalores y autovectores

El latido lineal de la naturaleza, la economía y la ingeniería

Imaginemos que una matriz no es solo una tabla de números, sino una transformación: estira, encoge, rota o voltea el espacio. Dentro de esa transformación hay direcciones privilegiadas, vectores que solo se alargan o acortan, sin cambiar su orientación. Esos vectores son los autovectores; el factor de estiramiento es el autovalor.

Este concepto, aparentemente abstracto, late bajo la piel de disciplinas muy concretas. En economía aparecen en modelos de crecimiento y en el análisis de insumo‑producto de Leontief (premio Nobel). En física son los estados estacionarios de la ecuación de Schrödinger y las frecuencias propias de una estructura vibrante. En ingeniería permiten entender la estabilidad de un puente o reducir la dimensionalidad de millones de datos (PCA). Con esta pequeña guía, desentrañaremos su esencia paso a paso, con ejemplos que caben en una servilleta.

Definición: la ecuación Av = λv

Sea A una matriz cuadrada de tamaño n×n. Decimos que un vector v (no nulo) es un autovector de A si existe un número (real o complejo) λ tal que:

A · v = λ · v

Ese número λ es el autovalor asociado al autovector v. Geométricamente, significa que la transformación A, aplicada a v, produce un vector paralelo a v (o antiparalelo si λ es negativo).

Ejemplo sencillo (matriz 2×2)

Matriz: A = ⎡2 1⎤ (escribimos por filas: [2,1; 1,2])
            ⎣1 2⎦

Tomemos v₁ = [1, 1] (vector fila, pero lo tratamos como columna). Entonces:

A·v₁ = [2·1 + 1·1 , 1·1 + 2·1] = [3, 3] = 3·[1, 1] = 3·v₁.

Por tanto, λ₁ = 3 es un autovalor y v₁ su autovector.

Probemos con v₂ = [1, -1]: A·v₂ = [2·1 + 1·(-1), 1·1 + 2·(-1)] = [1, -1] = 1·v₂.

Así, λ₂ = 1 es el segundo autovalor, con autovector v₂.

Un mismo autovalor puede tener múltiples autovectores (de hecho, todo el plano menos el origen si λ es el mismo para dos direcciones independientes). Pero empecemos con el cálculo sistemático.

Cálculo: el polinomio característico

Para obtener los autovalores de una matriz A, reagrupamos la ecuación Av = λv como:

(A − λI) v = 0

donde I es la matriz identidad. Para que exista un vector v no nulo, la matriz (A − λI) debe ser singular (determinante cero). Así llegamos a la ecuación característica:

det(A − λI) = 0

Ejemplo paso a paso con la misma matriz

A = [[2, 1], [1, 2]] → A − λI = [[2−λ, 1], [1, 2−λ]]

Determinante: (2−λ)(2−λ) − 1·1 = (2−λ)² − 1.

Igualamos a cero: (2−λ)² − 1 = 0 → (2−λ)² = 1 → 2−λ = ±1.

Casos:
• 2−λ = 1 → λ = 1
• 2−λ = −1 → λ = 3

Obtenemos los dos autovalores esperados: λ₁ = 3, λ₂ = 1.

Para hallar los autovectores resolvemos (A − λI)v = 0 con cada λ.

λ = 3: A−3I = [[−1, 1], [1, −1]]. El sistema: −x + y = 0 → y = x. Autovector: (1,1) (o cualquier múltiplo).

λ = 1: A−1I = [[1, 1], [1, 1]]. Ecuación: x + y = 0 → y = −x. Autovector: (1,−1).

Este procedimiento es general: primero raíces del polinomio característico, después resolver el sistema homogéneo.

Diagonalización: A = P D P⁻¹

Cuando una matriz A tiene suficientes autovectores (n autovectores linealmente independientes para una matriz n×n), podemos diagonalizarla. Esto significa encontrar una matriz invertible P (cuyas columnas son los autovectores) y una matriz diagonal D (con los autovalores en la diagonal) tales que:

A = P D P⁻¹   o equivalentemente   D = P⁻¹ A P

Diagonalizar simplifica enormemente los cálculos: elevar A a una potencia, resolver sistemas dinámicos, etc.

Ejemplo con nuestra matriz

Autovectores: v₁ = (1,1) para λ=3 ; v₂ = (1,−1) para λ=1.

Construimos P = ⎡1 1⎤ (columnas: primero v₁, luego v₂)
           ⎣1 −1⎦

La diagonal D = diag(3, 1) = ⎡3 0⎤
                       ⎣0 1⎦

Verifiquemos (opcional) que A = P D P⁻¹. Calculamos P⁻¹ para esta matriz simple:

P⁻¹ = (1/det(P)) · ⎡−1 −1⎤T? det(P) = (1)(−1) − (1)(1) = −2. Mejor escribamos la inversa conocida:
  P⁻¹ = (1/2)⎡ 1 1⎤
           ⎣ 1 −1⎦ (comprobación: P·P⁻¹ = I).

Efectivamente, P D P⁻¹ reproduce A. La diagonalización es una factorización poderosa.

No todas las matrices son diagonalizables (por ejemplo, si faltan autovectores, como en los bloques de Jordan). La condición necesaria y suficiente es que exista una base de autovectores.

Espacios propios (o autoespacios)

Para un autovalor λ, el espacio propio asociado es el conjunto de todos los vectores que satisfacen (A − λI)v = 0, es decir, el núcleo de (A − λI). Siempre contiene al autovector y al vector cero, y forma un subespacio vectorial.

Su dimensión (número de autovectores linealmente independientes para ese λ) se llama multiplicidad geométrica, y nunca supera la multiplicidad algebraica (la multiplicidad de λ como raíz del polinomio característico).

Ejemplo ilustrativo

Con la matriz A = [[2,1],[1,2]], para λ=3 resolvimos (A−3I)v=0:

Ecuación: −x + y = 0 → y = x. El espacio propio E₃ = { t·(1,1) | t ∈ ℝ }. Es una recta (dimensión 1).

Para λ=1: (A−1I)v=0 → x + y = 0 → y = −x. El espacio E₁ = { t·(1,−1) | t ∈ ℝ }, también dimensión 1.

Sumando ambas direcciones obtenemos todo ℝ², por eso la matriz es diagonalizable.

Imaginemos una matriz con un autovalor doble pero un solo autovector independiente (p. ej. [[2,1],[0,2]]). El espacio propio sería solo una recta, y la matriz no sería diagonalizable (solo podemos obtener su forma canónica de Jordan).

Importancia

Hemos recorrido el camino desde la definición intuitiva hasta herramientas concretas: la ecuación característica, el cálculo de autovectores, la diagonalización y los espacios propios. Con una matriz 2×2 sencilla hemos visto todo el proceso.

Pero el verdadero valor está en las aplicaciones: la mecánica cuántica es una historia continua de autovalores (observables) y autovectores (estados); el algoritmo PageRank de Google se basa en el autovector principal de una matriz gigante; la estabilidad de un ecosistema se analiza con los autovalores de su matriz de interacciones. Incluso la compresión de imágenes o el reconocimiento facial (PCA) utilizan esta descomposición.

Dominar los conceptos de autovalores y autovectores es abrir una puerta a las matemáticas aplicadas modernas. Y como has visto, todo arranca de la sencilla idea de buscar direcciones que la transformación sólo estira. El resto es álgebra lineal… y un poco de práctica.

Resumen rápido
• Autovector v ≠ 0 : A v = λ v
• λ se obtiene de det(A − λI)=0
• Para cada λ, los autovectores forman un espacio propio
• Si hay n autovectores independientes, A es diagonalizable: A = P D P⁻¹
